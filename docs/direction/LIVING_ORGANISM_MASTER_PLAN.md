## 1. North star

The Syndicate is a **living organism** — better, a **country**. Every figure is read live from
the chain; nothing is frozen; no forecasts. **Transparency is the product and the moat**, not
the marketing. Posture: *ask nothing · everything is here · don't trust, verify · it's an
experience — observe, then join if it suits you.* The core insight that ties the whole thing
together: **in crypto, lying is free; on-chain, it is impossible.** So we sell — and show —
only what can be proven.

## 2. The doctrine spine (non-negotiables locked this session)

- **Truth-first:** realized / verifiable **YES**; projected / yield / "what you'll earn" **NO**.
  Never render a figure that isn't a live chain read. Every public figure links to its proof.
- **Not a security:** *adoption not revenue · volume not profit · treasury not profit · not a
  yield dashboard.* No investment framing. Membership is a **purchase of access/recognition/
  services**, never an investment.
- **Recognition is multi-axis, never wealth.** Capital is ONE axis (Capital / Builder /
  Connector / Operator / Verifier / Historian / Steward…), **never the throne**. No financial
  multiplier / yield / better price for holders — ever.
- **Identity:** market-buy ≠ member. The seat is **binary** (one wallet = one seat; more SYN =
  more footprint/recognition, not more seats). Ladder: Wallet → Seat → Identity → Alias → Rank
  → Recognition → History → Legacy.
- **Introduction, not referral.** A member **introduces** another; *source wins over claims*
  (attribution read only from the verified on-chain source registry). The commercial "referral
  commission" is a **separate, PAUSED/gated** layer.
- **Privacy:** protocol default = **aggregate / address-safe** (no doxxing). The individual is
  seen only via **self-readback** or **opt-in self-publish** (a member choosing to share their
  own card/alias). Never expose a non-consenting member.
- **Memory:** nothing auto-promotes; **promotion = a human act**; verdicts are **identity- AND
  amount-blind**; no "the founder did X"; the Chronicle is a **solemn record, not a social feed**
  (no casino motion, no pagination, oldest-first).
- **Governance is banned.** Where the origin reserves a DAO/member-memory track, we reframe it
  as **permanently non-promoting recognition**, never "awaiting DAO ratification."

  Seats Around You) **→ Ownership** (Portfolio + Collector) **→ Momentum** (Progression +

- **The institutions of the country already have names:** Register = the census · Chronicle =
  the national history · Activity = the daily pulse · Explorer + labels = the records office /
  état civil · Recognition/rank = social standing · Alias/labeling = the passport.

## 4. Engagement psychology — healthy FOMO, healthy addictiveness

**Principle:** every healthy lever is **TRUE**; every toxic lever needs a **LIE**. Because
everything is on-chain, **every fake is instantly falsifiable** — so our own architecture
*forces* the honest path, and honest is the only durable path. **The honest IS the addictive.**

**Use — full throttle (all true):** momentum ("47 seats this chapter") · self-progression
(your reputation/receipts/rank growing) · recognition/status (the ordinal "#7", the OG badge,
tenure) · narrative/anticipation (a chapter filling, an era closing) · collection (receipts,
NFTs, your wall) · curiosity ("since you were away…").

**Refuse (all false):** fake scarcity / fake countdowns · **fake live** (the recency-truth
rule: verb + pulse encode real event AGE, never the pill) · financial-loss FOMO ("you're
missing GAINS") · a **richest-payer leaderboard** as the status axis · casino / slot loops.

**Show the money** (the economist + smart-contract hats): the **aggregate economy goes BIG and
proud** (treasury, volume, routing, burn, growth). Individual amounts are **already public and
immutable on-chain** → showing them exposes nothing, and every one is **provable** (verify
link) — so our money-display is *more* credible than a Web2 dashboard, not less. Show it — as
**proof**, never as a wealth ranking. **The only economic red line: realized/verifiable YES,
projected/yield NO.**

## 5. The ethical income model

**NOT** an AI agent that manages portfolios / swaps / advises — that is a regulatory minefield
and it detonates the anti-security spine (the "three horsemen": hallucination, resonance,
regulation). Instead: **sell WHO YOU ARE, PROVEN.** Identity / reputation / labeling services.

- **Address labeling** (Etherscan/Arkham model) — the strongest income stream. Pay to have an
  address labeled in our explorer (e.g. "Zealy" instead of `0x…`). **Must be VERIFIED**
  (proof of control) — never pay-to-impersonate; verification *is* the product. Recurring,
  compounding with explorer usage.
- **Aliases** (ENS-style) — sell a name / recognition / vanity. Tied to the seat, non-financial,
  not a tradeable asset.
- **Shareable cards ("OG")** — viral growth + ethical FOMO; consent-gated identity.
- **Verifiable reputation** — the crown: unfakeable on-chain standing (tenure + receipts +
  contributions + introductions), multi-axis, never wealth-ranking.
- **Guide premium** — a service tier (depth/convenience); the **free Guide stays fully
  truthful** (never put the truth behind a paywall).
- **White-label (post-MVP, separate business)** — sell the truth-first Guide / verification kit
  to other honest projects.

**Guardrails:** labels verified not impersonable · consent for identity display · never tied
to a financial return. Any paid identity service and any non-custodial "you-sign" action is
founder-gated (money decision only — §8-⑧: no legal gate).

## 6. The Guide (support + assistant) — see GUIDE_SUPPORT_ASSISTANT_DOCTRINE.md for detail

- **Hybrid, deterministic-first.** Default = deterministic (FAQ search + live figures +
  suggested questions + honest triage; drop the fake "1" badge). LLM = **capped optional
  escalation** for free-form questions, RAG-grounded, that **falls back to deterministic** on
  budget/limit.
- **Model:** Groq (free) primary + DeepSeek V4 Flash fallback — OpenAI-compatible (swap = one
  line), key **server-side only**, endpoint **isolated** from the `/api/protocol/reality` spine.
- **User-level aware:** visitor / holder / member — derived from **verified on-chain state**,
  reading only the connected wallet's own public facts. Reinforces "market-buy ≠ member."
- **Security stack:** input length cap + topic pre-filter (kills "free ChatGPT") · token-based
  rate-limit + global budget cap + **circuit-breaker → deterministic** · output forbidden-copy
  filter (never a banned term, never a figure outside injected context) · monitoring · anti-spam
  (session token, honeypot, escalate challenge). Blast radius capped: **no tools, no actions,
  no wallet** → worst case is embarrassing text, caught by the output filter.
- **Named "Guide,"** not the protocol AI Layer (PENDING). Later, doctrine-native roles:
  **Historian** (narrates on-chain history) + **Verifier** (teaches you to check it yourself —
  the anti-advisor).

**Build state:** `syndicate-os` has ONLY the proven backend `activity-heartbeat` read-model +
